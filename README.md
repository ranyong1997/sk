## 安装

```git
1 -- 下载
git clone https://github.com/ranyong1997/sk.git

2 -- 安装
使用pip安装： `pip install -r requirements.txt`
清华镜像源安装：pip install -i https://pypi.tuna.tsinghua.edu.cn/simple -r requirements.txt

3 -- 运行
python main.py
```
